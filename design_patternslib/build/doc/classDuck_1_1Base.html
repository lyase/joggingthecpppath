A Duck class

This is a base class( an interface) for ducks .
this project illustrates extending a class
crude implementation using inheritance may soon provide unwanted side effect such as decoy flying just by adding a fly method to the ducks.
now as shown in this library (ducky) using the strategy patern you get build-ins protection against such side effects plus a dynamic behaviour ( run time!!
